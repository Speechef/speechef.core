# U1.2 — User Dashboard with Progress Stats

## Status: Blocked (requires U1.1)

## Why
The home dashboard (`home/index.html`) shows placeholder content (hardcoded videos,
fake experts, stubbed streak). It should show real, personalized data for each user.

## What
- Recent game sessions with scores
- Total games played and best scores per game
- Current and longest streak
- Recommended next activity based on weakest game score
- Replace "Upload Video" placeholder with meaningful content or roadmap it for later

## Acceptance Criteria
- [ ] Dashboard shows real game history (last 5 sessions)
- [ ] Stats cards: total games, best score, current streak
- [ ] "Recommended" section points to the user's lowest-scoring game
- [ ] All placeholders either wired up or clearly labeled "Coming Soon"
