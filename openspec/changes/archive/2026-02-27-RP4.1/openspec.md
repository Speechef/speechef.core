# RP4.1 — Role Play Topic Suggestion Chips

## Status: Unblocked

## Problem
The roleplay setup screen has a free-text topic input but users often don't know what to type, leading to blank topics and generic sessions.

## Solution
Add a row of 3–4 clickable suggestion chips below the topic input for each mode. Clicking a chip fills the input with that suggestion. Uses the existing `MODE_META` examples from the hub page.
